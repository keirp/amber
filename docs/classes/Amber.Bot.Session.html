customPrompt

(
  • message
  • filter
  • callback
)

A custom prompt.

Parameters:

  • message String

    Message to send (usually asking the user a question). If message is a null, it won't send a message.

  • filter Function
    • session Session
    • message String
    • callback Function
      • success Boolean
        Is the message satisfactory?
      • value Object
        The filtered value
  • callback Function

    The callback is called with an argument containing the user's filtered response.

Example:

function filter(session, message, callback) {
    if (!isNaN(parseFloat(message))) {
        callback(true, parseFloat(message));
    } else {
        callback(false);
    }
}

selectionPrompt

(
  • message
  • options
  • [showMessage=true]
  • callback
)

A prompt asking the user to choose from a selection of choices. The user can respond by typing the index of the choice (indexed by 1) or by typing the choice itself. The choice with the shorted Levenshtein distance will be chosen.

Parameters:

  • message String

    Message to send (usually asking the user a question). If message is a null, it won't send a message.

  • options Array

    An array of strings representing the users options.

  • [showMessage=true] Boolean optional

    Show the options to the user?

  • callback Function

    The callback is called with an integer argument representing the index of the chosen option.
