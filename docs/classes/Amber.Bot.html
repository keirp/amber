addRequire

(
  • params
)

Add a requirement to the bot.

Parameters:

  • params Object
    • name String

      The name of the requirement

    • [requries] Session optional

      An array of requirements that need to be completed before this requirement can be fulfilled.

    • [filter] Function optional

      Called when a requirement is met externally.

      • value String
        The value passed in externally to meet the requirement.
      • session Session
      • value Object
      • callback Function
      • callback.success Boolean
        Whether this input can successfully meet the requirement.
      • callback.value Object
        The filtered version of the object.
    • toGet Function

      Called to retrieve the value of the requirement.

      • session Session
      • callback Function
      • callback.success Boolean
        Whether the value was retreived successfully. If false, exits the venture for the target.
      • callback.value Object
        The value of the requirement.

Example:

{
    name:'name',
    requires:[],
    filter:function(session, name, callback) {
        callback(true, name);
    },
    toGet:function(session, callback) {
        session.prompt('What is your name?', function(name) {
            callback(true, name);
        });
    }
}

addTarget

(
  • params
)

Add a target to the chat bot.

Parameters:

  • params Object
    • name String

      The name of the target

    • requires Array

      An array of Strings of the required information to reach this target.

    • [beforeRequire] Object optional

      Called before a requirement's toGet method is called. Each key is the name of the requirement and each value is a callback with the following arguments.

      • session Session
      • callback Function
        Call when done.
      • [callback.value] Object optional
        A value for the requirement. If a value is passed in, toGet will not be called.
    • [hasAccess] Function optional

      A function that returns whether the given session has access to this target.

      • session Session
    • onExecute Function

      Called when all requirements are met.

      • session Session
      • callback Function
        Call when the target is done executing.
      • [callback.next] String optional
        The next target to path to.

Example:

{
    name:'greeting',
    requires:['name'],
    onExecute:function(session, callback) {
        session.sendMessage('Hello there, ' + session.get('name'));
        callback();
    }
}

setTargetParser

(
  • parser
)

Set the method that turns messages into targets.

Parameters:

  • parser Function

    parser(message, callback) A callback that is passed a message. When the target has been identified, call the callback with the target as an argument.

Example:

bot.setTargetParser(function(message, callback) {
    if (message == 'hello') {
        callback('greeting', {});
    } else if (message == 'hello. my name is bob.') {
        callback('greeting', {'name':'bob'});
    } else {
        callback('none', {});
    }
});
